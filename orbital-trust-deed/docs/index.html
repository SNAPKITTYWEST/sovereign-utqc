Orbital Trust Deed

A mesh-facing verification layer for public orbital telemetry. CelesTrak, NASA GIBS, and NOAA GOES feeds are checked gate by gate, sealed into a WORM chain, then exposed to the agent mesh as governed action evidence.

$ verify orbital_feed --source CelesTrak
[01] SOURCE CelesTrak public TLE feed accepted
[02] SCHEMA required fields present
[03] TIMESTAMP DateTime verified
[04] FRESHNESS within latency window

$ seal trust_deed

Mesh Flow

01 · INGEST

Telemetry source

CelesTrak, NASA GIBS, or NOAA GOES emits public orbital state.

02 · VERIFY

Trust gates

The feed must pass source, schema, timestamp, and freshness gates.

03 · DEED

Trust Deed

A deed ID and deed hash are generated from verified feed evidence.

04 · SEAL

WORM Chain

The deed is written to an append-only SHA-256 chain.

05 · GOVERN

Agent gate

The mesh allows or blocks downstream action based on the sealed deed.
