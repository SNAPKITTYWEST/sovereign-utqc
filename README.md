### What I'm building

**[SnapKitty Sovereign OS](https://github.com/SNAPKITTYWEST/DEVFLOW-FINANCE)** — a financial operating system where the AI agents think out loud.

11 agents. Each one named, opinionated, with a domain and a peer network. Every decision goes through a three-pillar preflight, runs through a local LLM (Ollama), and gets sealed with SHA-256. The War Room UI shows every step as it happens — in real time.

The backend is not hidden. It is the product.

### The operating model

Most AI tooling keeps the machine in the background. You talk to it, it responds, you paste the output. The machine is invisible.

Here, the machine is the substrate. I am the Reverse Engineer — I read what the machine builds and understand the intelligence from the inside out. Between the machine writing and me reversing, a system gets built that neither could produce alone.

> *The machine is the extension of the man's mind.*
