METAMINE is not a programming language.

You are laying a minefield.

When you execute it, you are not running code.

## The Museum

**[→ Enter the Museum](https://snapkittywest.github.io/metamine/)**

The museum is an interactive experience where:

- WebGL shaders visualize mine ruptures in real-time
- p5.js canvases render generative art from execution traces
- You can place mines and watch them detonate
- Every execution leaves a cryptographic seal
